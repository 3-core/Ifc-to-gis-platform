# Ifc-to-gis-platform
## 프로젝트 환경
- Tool : Intellij Community
- Apache Tomcat
- JDK 17
- Spring Boot 2.7.4
- maven
- war


## 브랜치 관리
- main 브랜치
- develop 브랜치 : 각 기관에서 merge해주실 브랜치로 세 기관이 공유하는 브랜치입니다. <br>
  *반드시 rebase 후 merge 해주세요.
- 각 기관 브랜치(예시: 수성 브랜치 ss)
  - 기관 브랜치 내의 브랜치 관리 방식은 기관별로 결정하셔서 사용하시면 됩니다.
